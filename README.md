## Instrucciones

-Objetivo: derrotar a las naves enemigas
-Controles: la nave se mueve con las flechas de direccion y dispara con la tecla Z

## Reglas de Juego

-La nave comienza con 3 vidas
-cada impacto de proyectil enemigo quita una vida
-solo se puede mover de izquierda a derecha
